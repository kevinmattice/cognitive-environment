# Environment

An Environment is the broadest operating context in the atlas: the world or domain within which systems, interfaces, projects, and artifacts exist.

## When to use it

Use this type for the top-level context that contains many connected parts.

## Examples

- [[CCE]]

## Not to be confused with

Do not confuse an Environment with a single tool, host, or application. Those are usually better modeled as [[System]] or [[Infrastructure]].
