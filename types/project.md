# Project

A Project is a bounded body of work undertaken to build, improve, migrate, or explore something.

## When to use it

Use this type for active or planned work with direction and intent.

## Examples

- A future project to build or expand the atlas

## Not to be confused with

Do not confuse a Project with the artifact it produces or with a standing system that continues to operate after the work ends.
