# Capability

A Capability is something the environment can do.

## When to use it

Use this type for reusable functions, powers, or behavioral abilities that may be provided by one or more systems.

## Examples

- [[Memory]]
- [[Provenance]]

## Not to be confused with

Do not confuse a Capability with the specific system that implements it, or with an artifact that documents it.
