# Infrastructure

Infrastructure is the underlying substrate that hosts, carries, or enables systems and interfaces.

## When to use it

Use this type for foundational technical context such as hosts, platforms, or shared runtime layers.

## Examples

- [[Colossus]]
- [[Element]]

## Not to be confused with

Do not confuse Infrastructure with the higher-level systems or projects that run on top of it.
