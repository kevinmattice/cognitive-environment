# Artifact

An Artifact is a concrete thing produced, stored, or referenced in the environment.

## When to use it

Use this type for documents, registries, corpora, atlases, and other durable outputs.

## Examples

- [[Atlas]]
- [[Corpus]]
- [[Cognitive Registry]]

## Not to be confused with

Do not confuse an Artifact with the project that creates it or the capability it may support.
