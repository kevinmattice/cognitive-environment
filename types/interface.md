# Interface

An Interface is a point of interaction between people, agents, or systems.

## When to use it

Use this type when the main purpose of something is access, exchange, navigation, or interaction.

## Examples

- [[Tolaria]]
- [[GitHub]]

## Not to be confused with

Do not confuse an Interface with the broader system behind it or with the information artifact exposed through it.
