# Contribution Rules

Kevin, ChatGPT, Codex, and future CCE components should contribute to this atlas carefully and incrementally.

## Rules

- Make small changes.
- Use explicit relationships.
- Do not bulk import material without review.
- Prefer clarity over cleverness.
- Give every new entity a type and a status.
- Mark uncertain ideas as `Idea` or with status `Proposal` instead of treating them as operational fact.

## Working norm

This vault is an ontology seed. Add structure first, then grow content slowly as relationships and status become clearer.
