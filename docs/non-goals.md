# Non-Goals

Phase 1 of CCE explicitly does not include the following:

## Behavioral non-goals

- Autonomous agent behavior
- Multi-agent orchestration
- Background autonomous workflows
- Hidden memory systems

## Capability non-goals

- Web booking
- Live travel search
- Broad shell execution
- Arbitrary filesystem access
- Vector database integration
- Embeddings-based retrieval
- Docker orchestration expansion

## Process non-goals

- Shipping a Matrix bot just because plumbing is tempting
- Claiming end-to-end capability before verification exists
- Treating Hermes as reusable foundation without re-deriving contracts

## Reminder

If a future task seems useful but weakens boundedness or inspectability, it should be treated as out of scope until explicitly re-justified.
