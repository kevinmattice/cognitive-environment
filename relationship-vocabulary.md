# Relationship Vocabulary

This note defines the first small set of relationship labels used in the atlas.

## belongs_to

Use when an entity is part of a larger parent context or environment.

## contains

Use when one entity directly includes or holds other entities.

## supports

Use when an entity helps enable another entity's operation or use.

## uses

Use when an entity actively relies on another entity as a tool, interface, or dependency in practice.

## depends_on

Use when an entity requires another entity in order to function.

## governs

Use when a note, decision, or system sets rules or boundaries for another thing.

## represents

Use when one entity stands in for, models, or expresses another thing.

## related_to

Use for meaningful connections that are real but not yet specific enough for a stronger label.

## replaces

Use when one entity is intended to succeed or supersede another.

## feeds

Use when one entity provides material, data, or input into another.
